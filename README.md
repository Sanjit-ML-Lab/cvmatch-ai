# cvmatch-ai
Build a smart CV-matching web app that accepts job descriptions and CV uploads - Extracts keywords and computes match scores - suggest changes needed


Business Requirements Document (BRD)
 
Objective
Build a smart CV-matching web app that:
- Accepts job descriptions and CV uploads
- Extracts keywords and computes match scores
- Provides visual feedback (progress rings, keyword cloud, dashboard)
- Supports authentication and persistent history
- Exports reports
 
Core Features
- Dual input interface (textarea + file upload)
- Real-time keyword extraction
- CV parsing (PDF/DOCX/TXT)
- Match scoring (strong, partial, gaps)
- Results dashboard with visual indicators
- Authentication (sign-up/sign-in)
- History page with saved analyses
- Export report (PDF/TXT)
 
Non-Functional Requirements
- Responsive design
- Secure authentication
- Scalable backend
- MySQL persistence
- AWS-ready deployment
